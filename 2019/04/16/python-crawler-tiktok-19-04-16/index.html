确保电脑安装了 charles / fiddler / wireshark 或其他能抓包的工具，然后在手机或者模拟器上配置好代理，记得安装证书，接着开始抓包。

注：最好在 Android7.0 以下配置，7.0及以上版本不能抓https了，有点麻烦，推荐使用模拟器
